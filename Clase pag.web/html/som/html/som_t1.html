El Sistema Informático

Hay diferentes tipos de ordenadores:

  • Super ordenadores: Se pueden realizar cálculos que son prácticamente imposibles o que se harían muy lentamente con otros. Es un dispositivo informático y no está accesible al público.
  • Macroordenadores: Están entre los superordenadores y servidores, que son grandes y pueden dar servicio a otros.
  • Servidores o estaciones de trabajo: Son ordenadores potentes, que pueden dar servicios a otros ordenadores dentro de una red. Puede ocupar tanto como dependiendo de la potencia.
  • Ordenadores personales o PC: se utilizan tanto a nivel doméstico como profesional. Son los menos potentes y surgieron después del microprocesador.

Informática: Rama de la Ingeniería que estudia el hardware, las redes de datos y el software necesarios para tratar información de forma automática.

  • Tarjetas perforadas: pieza de cartulina que contiene información digital.
  • Máquina Analítica (Charles Babbage): capaz de calcular tablas numéricas.
  • Máquina tabuladora de tarjetas perforadas: se utilizaron para meter datos en ordenadores.

Generaciones de ordenadores

  1. Primera generación (1940-1959): (ENIAC) estos eran muy grandes y poco óptimos, con un alto consumo energético, está compuesto por tubos de vacío. El UNIVAC 1 es el primer ordenador con fines comerciales. Aparece el primer lenguaje de programación (lenguaje ensamblador).
  2. Segunda generación (1960-1965): empezaron a tener transistores y se vuelven más pequeños y económicos. Aparece el concepto de periféricos.
  3. Tercera generación(1966-1971): empezaron a contener los circuitos integrados. Se vuelven más pequeños y económicos, al igual que aumenta el consumo de estos.
  4. Cuarta generación(1971-1980): tienen microprocesadores, se reduce el tamaño del ordenador y surge el concepto de ordenador personal.
  5. Quinta generación (1981-actual): hacen el primer PC.

Componentes del Hardware

Elementos físicos

  • Carcasas
  • Periféricos: entrada, salida ; entrada y salida.
  • Memorias auxiliares: discos duros, USB
Elementos funcionales:
  • Unidad central de proceso (UCP)
    • Registro contador de Programa: contiene la dirección de la instrucción siguiente que hay que leer de la memoria.
    • Decodificador: transforman un código de tipo binario en código decimal.
    • Generador: rutina especial que es usada para controlar el comportamiento de iteración en un "loop".
    • Reloj: genera señales para controlar y sincronizar todo el sistema.
  • Unidad aritmético-lógica (UA)
    • Registros de datos: datos de entrada u operandos.
    • Registro acumulador (AC): resultados.
    • Registro de datos (RE): almacena las condiciones.
  • - Memoria principal (MP) - RAM (Random Access Memory) - Registro de direcciones - Registro de intercambio - Selector de memoria - ROM (Read Only Memory)
  • Buses
    • Bus de Datos: información (datos/ordenes)
    • Bus de Dirección: a dónde
    • Bus de Control: para ver si está bien
  • Periféricos
  • Unidad Entrada/Salida
