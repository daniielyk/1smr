COMPUTER SYSTEM CONCEPT

A computer is an electronic device, capable of receiving information, storing it for a period of time and processing it to offer the result abroad.

  • Accept and transmit information: through the input/output devices, the computer receives and communicates with the exterior. These devices encode the input information to a format that the computer understands and also encode it for transmission abroad.
  • Data storage: this function is carried out on storage devices, of which the computer has a great variety, depending on their nature and their specific purpose.

Computers can be classified as one of three types: a supercomputer, a general-purpose computer, or a special-purpose computer.

  • A supercomputer is a computer with high performance compared to a general-purpose computer, often used in research, scientific, or military environments.
  • A server is a computer that accepts and responds to requests made over a network. The device that makes the request and receives a response from the server is called a client.

HISTORICAL BACKGROUND OF COMPUTERS

The history of computing is longer than the history of computing hardware and modern computing technology. In ancient times, methods for mathematical calculations were known. In the 17th century, calculating machines began to be invented.

Generations of computers

Analytical Engine, in which the basic framework of today‘s computers is based on. Generally speaking, computers can be classified into five generations.

  1. First Generation of Computer: Vacuum tubes and boards. In the mid-1940s, Howard Aiken at Harvard and John von Neumann at Princeton, among others, managed to build calculating machines.
  2. Second Generation of Computer: Transistors. Replaced vacuum tubes, smaller, and consumed less electricity. Communication with these computers is through high-level languages or programming languages.
  3. Third Generation of Computer: Integrated circuits. Emerged with the development of integrated circuits, leading to the invention of the microprocessor.
  4. Fourth Generation of Computer: Characterized by the appearance of the microprocessor. The age of the personal computer arrived.
  5. Fifth Generation of Computer - 2019 - Quantum Computers: IBM presented its first commercial quantum computer, called the IBM Q System One, in 2019.

COMPUTER COMPONENTS

Any kind of computer consists of HARDWARE and SOFTWARE.

Hardware

Computer hardware is the collection of physical elements that constitute a computer system, all of which are physical objects that can be touched.

  • Computer case: Contains the central processing unit (CPU) and other essential components.
  • Peripheral device: Hardware device that isn’t required for the computer's primary function. Includes input, output, and input/output devices.
    • Input devices: Used to enter information into a computer system.
    • Output devices: Used to send data from a digital device, such as a computer, to a user or another device.
  • Storage devices: Used to store application and user data, such as hard disk drives and USB flash drives.

Modern computers are based on a stored-program concept introduced by John Von Neumann. The basic structure is like Von Neumann architecture.

Buses: The types of computer buses are the address bus, the data bus, and the control bus.

  • The address bus: A one-way pathway that carries information about where data is stored in memory.
  • The data bus: A two-way pathway carrying the actual data to and from the main memory.

What is a CPU?

The central processing unit is the brain of the computer that processes program instructions.

Arithmetic logic unit and Registers.

  • Registers are small amounts of high-speed memory contained within the CPU. They are used to store small amounts of data needed during processing.
  • The control unit of the central processing unit regulates and integrates the operations of the computer. It decodes and executes instructions.

What is a Main Memory Unit?

RAM is the hardware in a computing device where the operating system, application programs, and data in current use are kept for quick access by the device's processor. RAM is usually associated with DRAM, a type of memory module.

What is an Input/Output Unit?

The input/output unit communicates the microprocessor with components such as auxiliary memories, external storage units, or peripherals.

Computer software falls into three broad classes:

  • System software: The principal system software is the operating system, managing hardware, data, program files, and other system resources, providing means for user control via a graphical user interface.

Software licenses

A software license is a contract between the entity that created and supplied an application, source code, or related product and its end user.

  • Proprietary – Also referred to as "closed source."
  • "Free software" means software that respects users' freedom and community.

While open source is a development philosophy that is more business-oriented, free software is a social and moral philosophy. That’s why the term open source is more pleasant to the corporate world because it places less emphasis on freedom.

Why are we going to learn the numbering systems?

The computer is an electronic device. In fact, any system that can represent at least two states can represent information. To represent a number, we use the binary system, not the decimal number system that we use in everyday life.

Binary system

  • The base of the binary number system is 2, because it has only two digits.
  • The binary system is the one that the computer would use internally.
  • Each binary digit is also called a bit. Binary number system is also a positional value system, where each digit has a value expressed in powers of 2.

8 bits = 256 values

Octal Number System

Octal Number System is a type of number system that has a base of eight and uses digits from 0 to 7.
