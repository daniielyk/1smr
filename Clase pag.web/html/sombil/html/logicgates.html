Logic Gates Summary

Logic gates are fundamental building blocks in digital circuits. They perform basic operations on binary inputs (0 or 1) to produce binary outputs. Here's a summary of common logic gates:

Gate Symbol Description
AND & Outputs true (1) only if both inputs are true.
OR | Outputs true (1) if at least one input is true.
NOT ¬ Outputs the opposite of the input; negates the input.
XOR ⊕ Outputs true (1) if the number of true inputs is odd.
NAND ⊼ Outputs false (0) only if both inputs are true.
NOR ⊽ Outputs false (0) if at least one input is true.
