Estructura básica de una página web

Una página web básica se compone por: html, head y body.

Html: Indica que la estructura en árbol de este lenguaje tiene su raíz en el elemento HTML.
Head: Delimita la cabecera del documento. Aquí podemos poner el título del documento o el sistema de escritura (inglés, español, etc.) que vamos a utilizar, entre otros.
Body: Es el contenido de la página web.

A parte de estas tres etiquetas, en un documento es recomendable añadir <!DOCTYPE html> para especificar qué tipo de documento estamos creando.

Etiquetas

<p>  Crea un párrafo

<h1>  Crea títulos. Hay desde h1 hasta h6, yendo desde el más grande hasta el más pequeño.

<br />  es una etiqueta "especial", ya que solo se cierra. Sirve para añadir espacios o intros en nuestro texto.

<hr />   Al igual que la anterior, esta etiqueta solo se cierra. Inserta una línea horizontal.

<pre>  Con esta etiqueta el navegador visualizará el texto tal y como aparece. Respetará espacios y etiquetas entre otros.

<b> o <strong>  Pone el texto en negrita.

<i> o <em>  Pone el texto en cursiva.

<sub>   Pone el texto en subíndice.

<sup>   Pone el texto en superíndice.

<del>  Tacha el texto.
