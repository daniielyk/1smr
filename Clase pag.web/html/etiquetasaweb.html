Estructura básica de una página web

Una página web básica se compone por: html, head y body.

Html: Indica que la estructura en árbol de este lenguaje tiene su raíz en el elemento HTML.
Head: Delimita la cabecera del documento. Aquí podemos poner el título del documento o el sistema de escritura (inglés, español, etc.) que vamos a utilizar, entre otros.
Body: Es el contenido de la página web.

A parte de estas tres etiquetas, en un documento es recomendable añadir <!DOCTYPE html> para especificar qué tipo de documento estamos creando.

Etiquetas

<p>  Crea un párrafo

<h1>  Crea títulos. Hay desde h1 hasta h6, yendo desde el más grande hasta el más pequeño.

<br />  es una etiqueta "especial", ya que solo se cierra. Sirve para añadir espacios o intros en nuestro texto.

<hr />   Al igual que la anterior, esta etiqueta solo se cierra. Inserta una línea horizontal.

<pre>  Con esta etiqueta el navegador visualizará el texto tal y como aparece. Respetará espacios y etiquetas entre otros.

<b> o <strong>  Pone el texto en negrita.

<i> o <em>  Pone el texto en cursiva.

<sub>   Pone el texto en subíndice.

<sup>   Pone el texto en superíndice.

<del>  Tacha el texto.

Marcado avanzado de caracteres

<abbr>   permite indicar el significado de una abreviatura.

<acronym>   permite indicar el significado de un acrónimo.

<dfn>   permite indicar la explicación ded una definición.

Estas etiquetas deben llevar el atributo title en el que escribiremos el significado de las palabras.
Ejemplo: <abbr title= "Su Alteza Real">SAR</abbr>  

<cite>   marca un texto como cita literal. Suele aparecer en cursiva.

<cod>   permite indicar que el texto al que engloba pertenece a código fuente de un lenguaje de programación.

<samp>   indica un ejemplo de salida por pantalla de un programa informático.

<kbd>   se usa para indicar que el texto incluido indica una tecla del teclado o una combinación de teclas. Utiliza el atributo title

<var>   indica nombres de variables.

<time>   permite indicar una hora.

<bdo>   Indica la dirección del texto. Utiliza el atributo dir

Marcado avanzado de párrafos

<blockquote>   Permite indicar una cita de texto extensa. Utiliza el atributo cite

Correcciones

<ins>   Indica que el texto marcado con este elemento es provisional. Utiliza los atributos cite para añadir una URL que lleva al texto original. y datetime indica la fecha de la modificación con cuatro cifras para el año, dos para el mes y dos para el día.
